Listing repositories, star count and used languages of a GitHub user

This web app allows you to print the repositories, total stargazers and used languages of a selected GitHub user.
The repositories with corresponding star amounts and the total number of stargazers are always printed.
Listing the languages takes significantly more time than just listing repositories, therefore it is optional.

Below, provide the username you want to check and determine if you want to see the most used languages (type "y" for yes and "n" for no).
Depending on the number of repositories that the selected user has published on his profile, this action can take from several seconds up to several minutes.


WARNING
Please note that there is an hourly limit imposed by GitHub on the API access.
After the limit is reached, only 30 first repositories together with the starcount in them will be shown.
The list of used languages will only contain languages used in those 30 repositories and will be printed without specifying their usage in bytes.
Moreover, the statistics for organisation accounts (like Microsoft) will not be presented correctly.
